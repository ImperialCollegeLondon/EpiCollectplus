2. Data Visibility

You can choose whether your data is available to anyone via this website, or restricted to the project managers and curators. You will be able to add Curators and Managers from the 'Manage Project' page once your project is created.

3. Project Listing on the Homepage

You can choose to have projects listed on the front page of the website, and if someone uses the APIs to get a list of projects. This may help to publicise your project and encourage people to submit data. If you choose to do this then consider adding a description and project image through the 'Manage Project' page.

4. Project Managers

Managers have the ability to add and remove other project managers and curators, they can also update the project settings and the project definition. Managers can also Add, edit and delete data using the web interface.
